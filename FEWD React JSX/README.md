This is a simple React application that allows users to increment and decrement a counter. The app demonstrates basic React concepts such as state management using the useState hook, JSX syntax, and event handling.

**Features**
- Displays a counter value that starts at 0.
- Provides buttons to increment (+) and decrement (-) the counter value.
- Updates the displayed counter value dynamically as the buttons are clicked.

**Usage**
- The app shows a counter initialized at 0.
- Clicking the + button increases the counter value by 1.
- Clicking the - button decreases the counter value by 1.


**Commands**
- use `npm run dev` for starting the server.
- use `npm run test:serve` for viewing test cases.

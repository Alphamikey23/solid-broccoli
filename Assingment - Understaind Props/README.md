# React Props Lab

## **Objective**

In this lab, you will learn how to work with **props** in React by passing data from `DataComponent.jsx` to both `App.jsx` (functional component) and `AppClass.jsx` (class component). Your goal is to render an image gallery and pass the provided test cases.


---

## **Instructions**

1. **DataComponent.jsx**
   - The image data is already added in `DataComponent.jsx`.

2. **Pass Data as Props**
   - Import `imageData` from `DataComponent.jsx` into `App.jsx` and `AppClass.jsx`.
   - Pass this data as props to render the images.

## **Test Cases**
The application should pass these test cases:
- **App Component**: renders the gallery title and images correctly.
- **AppClass Component**: renders the gallery title and images correctly.
